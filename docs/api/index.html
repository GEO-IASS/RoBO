class robo.solver.contextual_bayesian_optimization.ContextualBayesianOptimization(dims_Z=None, dims_S=None, context_fkt=None, acquisition_fkt=None, model=None, maximize_fkt=None, S_lower=None, S_upper=None, objective_fkt=None, save_dir=None, num_save=1)[source]¶

Class for contextual Bayesian optimization. Adds an context function, which obtains a context vector Z before each iteration. The objective is then to optimize for this given context.

For details see [Contextual Gaussian Process Bandit Optimization. 2011. Krause, Andreas and Cheng S. Ong. Advances in Neural Information Processing Systems 24.]
